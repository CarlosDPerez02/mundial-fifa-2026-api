API REST desarrollada con FastAPI, SQLAlchemy y PostgreSQL para gestionar jugadores y selecciones del Mundial FIFA 2026.

Tecnologías utilizadas:
Python, 
FastAPI,
SQLAlchemy,
PostgreSQL,
Docker,
Uvicorn,
Pydantic.


Funcionalidades:
Registrar jugadores,
Listar jugadores,
Buscar jugadores por ID,
Buscar jugadores por nombre,
Actualizar nombre, selección, goles y asistencias,
Eliminar jugadores,
Consultar goleadores,
Consultar jugadores por participaciones de gol y asistencias.


Ejecutar el proyecto:
pip install -r requirements.txt
uvicorn main:app --reload

La documentación de la API estará disponible en:
http://127.0.0.1:8000/docs
